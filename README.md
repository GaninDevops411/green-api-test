# GREEN API Test

Тестовое задание на должность DevOps.

## Реализовано

HTML-страница с вызовами методов GREEN-API:
- getSettings
- getStateInstance
- sendMessage
- sendFileByUrl

## Функционал

На странице доступны:
- ввод `idInstance`
- ввод `ApiTokenInstance`
- отправка текстового сообщения
- отправка файла по URL
- отображение ответа API в отдельном поле только для чтения

## Как запустить локально

1. Скачайте проект
2. Откройте файл `index.html` в браузере

## Как использовать

1. Создайте инстанс в GREEN-API
2. Подключите номер телефона через QR-код
3. Введите:
   - `idInstance`
   - `ApiTokenInstance`
4. Используйте нужные методы на странице
